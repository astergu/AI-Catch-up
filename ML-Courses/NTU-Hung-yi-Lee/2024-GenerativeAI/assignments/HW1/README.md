# GenAI HW1：真假难辨的世界

## 目标

- 体验生成式AI的能力和限制

## 任务

判断提供的内容是否由AI生成。
- 覆盖的内容形态：语言，歌声，文本，图片和视频
- 一共40个问题

## 数据

- NTU Cool Quiz 
- [https://drive.google.com/drive/folders/1IqrfovQbPj9nuRBxQ80LcocqkUFxYNfL](https://drive.google.com/drive/folders/1IqrfovQbPj9nuRBxQ80LcocqkUFxYNfL)
